CPUImportantly

The main component of a computer, designed to control all its devices and perform arithmetic and logical operations on data..

Fan Importantly

A device consisting of a fan and a radiator. Needed to cool the processor, as well as remove hot air from the desktop.

Motherboard Importantly

The main board, which contains the main components of the computer that provide logic. The functionality of the computer depends on the motherboard, which is determined by the elements of the board itself and the devices connected to it..

GPU Importantly

Designed to generate and process images and then display them on the screen of a peripheral device

RAM Importantly

High-speed computer memory designed to write, store and read information during processing.

Network card

A Network interface card (also known as a NIC, network card, or network interface controller) is an electronic device that connects a computer to a computer network, usually a LAN. It is considered a piece of computer hardware. ... Network cards let a computer exchange data with a network.

Drive Importantly

Permanent computer storage device that provides long-term data storage.

SSD

Computer storage device based on memory chips and their control controller, which does not contain moving mechanical parts.

Power Importantly

Designed to provide computer components with DC power. Its task is to convert the mains voltage to the specified values ​​of the supply voltage, its stabilization and protection against minor interference from electrical supply networks

Case Importantly

Physically, it is a basic supporting structure, which is designed for further filling with hardware to create a computer.

OS

Basic set of programs that controls the hardware component of a computer or virtual machine; provides control of the computing process and organizes interaction with the user.
